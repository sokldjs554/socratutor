# 가정법 (Conditionals)

## 형태
- 가정법 과거: If + 과거동사, 주어 + would/could + 동사원형
- 가정법 과거완료: If + had p.p., 주어 + would/could + have p.p.

## 핵심 개념
**현재 사실의 반대**를 가정하면 과거형을, **과거 사실의 반대**를 가정하면 과거완료형을 쓴다.
시제가 한 단계씩 뒤로 밀리는 것이 핵심.

## 용법
1. If I **were** rich, I **would buy** a house. (지금 부자가 아님 — 현재 반대)
2. If I **had studied** harder, I **would have passed** the exam. (과거에 공부 안 했음 — 과거 반대)
3. 단순 조건(가정법 아님): If it **rains** tomorrow, I **will stay** home. (실제 가능한 일)

## 자주 하는 실수
- ❌ If I would have money, I would buy it.
- ✅ If I **had** money, I would buy it. (if절에는 would를 쓰지 않는 것이 원칙)
- ❌ If I studied harder, I would have passed. (시제 불일치 혼합)
- ✅ If I **had studied** harder, I **would have passed**. (과거 반대는 양쪽 다 완료형)
